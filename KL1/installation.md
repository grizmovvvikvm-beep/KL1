# Installation Guide

## System Requirements

- CentOS 7/8 or RHEL 7/8
- 2GB RAM minimum
- 10GB free disk space  
- Root access required

## Quick Installation

```bash
# Clone the repository
git clone https://gitlab.com/your-username/kurs-light-vpn.git
cd kurs-light-vpn

# Run the installer
sudo ./scripts/install.sh
